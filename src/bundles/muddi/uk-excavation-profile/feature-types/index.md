# Feature types

Utility feature tables grouped by utility domain. Each domain repeats the same ~15 MUDDI feature roles (network, node, link, site, service area, accessories…). Every table is **flattened** — all inherited attributes are inlined, so a feature is one wide standalone table queried without joins.

* [Drainage](drainage/index.md) - drainage feature types
* [Electricity](electricity/index.md) - electricity feature types
* [Fuel & Chemicals](fuelandchemicals/index.md) - fuelandchemicals feature types
* [Gas](gas/index.md) - gas feature types
* [Sewer](sewer/index.md) - sewer feature types
* [Telecommunications](telco/index.md) - telco feature types
* [Thermal](thermal/index.md) - thermal feature types
* [Transport Signal](transportsignal/index.md) - transportsignal feature types
* [Water](water/index.md) - water feature types
* [Shared / Cross-domain](shared/index.md) - shared feature types
