**Abstract.** Type of NetworkAccessory that contains one or more NetworkConveyances. As a MUDDIobject, a Container may also contain other containers (e.g. nested ducts) or other NetworkAccessories, etc.

Specialises [NetworkAccessory](network-accessory.md).

This class defines no attributes of its own; it carries only those it inherits.
